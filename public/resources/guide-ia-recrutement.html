Guide IA et recrutement : 20 prompts

L'IA generative est un accelerateur pour les recruteurs. Ce guide propose 20 prompts concrets, testes et optimises, pour les principales taches du quotidien : redaction d'offres, sourcing, qualification, messages d'approche et analyse.

Rappel ethique : L'IA est un outil d'aide, pas un decideur. Ne l'utilisez jamais pour prendre une decision de recrutement automatique. Relisez et adaptez toujours les outputs. Respectez le RGPD et informez les candidats si vous utilisez des outils d'IA dans votre process.

A. Redaction d'offres d'emploi (Prompts 1-5)

Prompt 1 — Generer une offre d'emploi complete

Quand : Vous devez rediger une offre a partir d'un brief de poste.

Tu es un expert en redaction d'offres d'emploi pour le marche francais. Redige une offre d'emploi pour le poste de [POSTE] chez [ENTREPRISE]. Contexte : - Secteur : [SECTEUR] - Taille : [NOMBRE] salaries - Localisation : [VILLE], [POLITIQUE TELETRAVAIL] - Experience requise : [X] ans Structure attendue : 1. Accroche impactante (2-3 lignes, pas de cliche) 2. Presentation de l'entreprise (4-5 lignes) 3. Missions (5-7 bullet points, verbes d'action) 4. Profil recherche : must-have (3-4) et nice-to-have (2-3) 5. Ce qu'on offre (5-6 avantages concrets) 6. Process de recrutement (etapes + duree) Contraintes : - Ecriture inclusive (sans points medians) - Ton professionnel mais humain - 400-600 mots maximum - Pas de jargon corporate vide

Prompt 2 — Ameliorer une offre existante

Analyse cette offre d'emploi et propose des ameliorations : [COLLER L'OFFRE] Evalue sur ces criteres (note /10 chacun) : 1. Clarte des missions 2. Attractivite de l'accroche 3. Equilibre must-have / nice-to-have 4. Transparence sur le package 5. Ton et inclusivite Propose une version amelioree avec les changements expliques.

Prompt 3 — Generer des variantes d'accroche

Genere 5 accroches differentes pour une offre de [POSTE] chez [ENTREPRISE]. Styles : 1. Orientee mission / impact 2. Orientee defi / challenge 3. Orientee culture / equipe 4. Orientee croissance / apprentissage 5. Orientee marche / innovation Chaque accroche : 2-3 lignes, pas de superlatifs creux.

Prompt 4 — Adapter une offre a un canal specifique

Adapte cette offre d'emploi pour [CANAL] : [COLLER L'OFFRE] Canal : [LinkedIn / Welcome to the Jungle / Indeed / Twitter] Contraintes par canal : - LinkedIn : 150 mots max, emoji moderes, hashtags pertinents - WTTJ : structure imposee, ton decontracte - Indeed : SEO-friendly, mots-cles du metier - Twitter : 280 caracteres, lien vers l'offre complete

Prompt 5 — Optimiser le SEO d'une offre

Analyse cette offre d'emploi et optimise-la pour le SEO emploi : [COLLER L'OFFRE] Fournis : 1. Les 10 mots-cles principaux a integrer (intitule, competences, outils) 2. Un titre SEO optimise (60 caracteres max) 3. Une meta-description (155 caracteres max) 4. Les variations de l'intitule de poste a cibler 5. La version optimisee de l'offre

B. Messages de sourcing (Prompts 6-10)

Prompt 6 — Message d'approche LinkedIn personnalise

Redige un message d'approche LinkedIn pour ce candidat : Profil : - Nom : [PRENOM] - Poste actuel : [POSTE] chez [ENTREPRISE] - Element notable : [PROJET, ARTICLE, CERTIFICATION] Poste a pourvoir : - Titre : [POSTE] - Entreprise : [ENTREPRISE CLIENTE] - Proposition de valeur principale : [1 PHRASE] Contraintes : - Maximum 300 caracteres (InMail court) - Personnalise avec l'element notable - 1 seul call-to-action - Ton direct et respectueux, pas commercial - Pas de "je me permets de vous contacter"

Prompt 7 — Sequence de 3 messages de relance

Cree une sequence de 3 messages de relance pour un candidat qui n'a pas repondu au premier message. Contexte du poste : [POSTE] chez [ENTREPRISE] Premier message envoye : [RESUMER EN 1 LIGNE] Message 2 (J+4) : Angle different, apporter une info nouvelle (culture, projet, equipe) Message 3 (J+8) : Derniere relance, court et direct, proposer une alternative (email, appel) Chaque message : 2-4 lignes max. Ton : curieux, pas insistant.

Prompt 8 — Message pour un profil passivement en recherche

Redige un message d'approche pour un profil senior qui n'est pas en recherche active. Profil : [POSTE ACTUEL] chez [ENTREPRISE], [X] ans d'experience Element d'accroche : [CE QUI LE DIFFERENCIE] L'angle ne doit PAS etre "j'ai une opportunite". Utilise plutot : partage de contenu, invitation a un event, echange d'expertise, question sur son domaine. Objectif : creer une relation avant de pitcher le poste.

Prompt 9 — Email d'approche cold

Redige un email d'approche cold pour recruter un(e) [POSTE]. Infos : - Destinataire : [PRENOM], actuellement [POSTE] chez [ENTREPRISE] - Entreprise qui recrute : [NOM], [SECTEUR], [TAILLE] - USP du poste : [1-2 PHRASES] - Package : [FOURCHETTE] Format : - Objet : < 6 mots, personnalise - Corps : < 100 mots - 1 CTA clair (proposer un call de 15 min) - PS optionnel avec un lien (article, video, page carriere)

Prompt 10 — Recherche booleenne LinkedIn

Genere 5 requetes booleennes pour sourcer un(e) [POSTE] sur LinkedIn. Criteres : - Experience : [X-Y] ans - Competences cles : [LISTE] - Secteur : [SECTEUR] - Localisation : [ZONE] - Entreprises cibles : [LISTE OU TYPE] Fournis : 1. Requete large (volume) 2. Requete ciblee (precision) 3. Requete par entreprise cible 4. Requete par competence technique 5. Requete par titre alternatif Format : syntaxe booleenne LinkedIn (AND, OR, NOT, guillemets)

C. Qualification et analyse de CV (Prompts 11-15)

Prompt 11 — Analyser un CV vs un brief de poste

Compare ce CV avec le brief de poste ci-dessous. Brief de poste : [COLLER LE BRIEF - must-have et nice-to-have] CV : [COLLER LE CV OU LE RESUME] Fournis : 1. Tableau de correspondance must-have (oui/non/partiel) 2. Tableau de correspondance nice-to-have 3. Points forts du profil 4. Points d'attention / gaps 5. 3 questions specifiques a poser en screening 6. Recommandation : shortlist / screening / rejet

Prompt 12 — Preparer des questions d'entretien specifiques

A partir de ce CV, genere 8 questions d'entretien ciblees : CV : [COLLER LE CV] Poste : [POSTE A POURVOIR] Pour chaque question : - La competence evaluee - La question (format STAR) - Ce qu'on cherche dans la reponse (signaux positifs) - Les red flags a surveiller 4 questions sur l'experience passee 2 questions sur les competences techniques 2 questions sur le comportement / soft skills

Prompt 13 — Synthetiser un profil candidat

A partir de ces informations, redige une fiche synthese candidat a presenter au hiring manager : CV : [COLLER] Notes de screening : [COLLER] LinkedIn : [INFOS CLES] Format de la fiche : - Resume executif (3 lignes) - Experience cle (2-3 realisations chiffrees) - Adequation au poste (must-have : ok/ko) - Motivations identifiees - Points d'attention - Disponibilite et pretentions - Recommandation argumentee

Prompt 14 — Comparer 3 candidats finalistes

Compare ces 3 candidats finalistes pour le poste de [POSTE] : Candidat A : [RESUME] Candidat B : [RESUME] Candidat C : [RESUME] Criteres de comparaison : 1. [MUST-HAVE 1] 2. [MUST-HAVE 2] 3. [MUST-HAVE 3] 4. Culture fit 5. Potentiel de croissance 6. Risque (preavis, contre-offre, autres process) Fournis un tableau comparatif et une recommandation argumentee.

Prompt 15 — Debrief d'entretien structure

A partir de ces notes d'entretien brutes, genere un debrief structure : Notes : [COLLER LES NOTES] Format : 1. Resume de l'entretien (3-4 lignes) 2. Evaluation par competence (leadership, technique, collaboration, resultats) avec note /5 3. Points forts (3 max) 4. Points de vigilance (2-3 max) 5. Reponse aux questions cles du brief 6. Recommandation : GO / NO GO / A APPROFONDIR 7. Questions pour le prochain entretien (si applicable)

D. Productivite et process (Prompts 16-20)

Prompt 16 — Rediger un refus bienveillant

Redige un email de refus pour un candidat a l'etape [ETAPE] du process. Contexte : - Prenom : [PRENOM] - Poste : [POSTE] - Raison principale : [RAISON - sans etre discriminant] - Point positif : [CE QUI A ETE APPRECIE] Contraintes : - Ton empathique et respectueux - Donner un feedback actionnable (1-2 lignes) - Encourager le candidat - Laisser la porte ouverte pour le futur - 8-12 lignes maximum

Prompt 17 — Generer un plan d'onboarding

Cree un plan d'onboarding 30-60-90 jours pour un(e) [POSTE] chez [ENTREPRISE]. Contexte : - Taille de l'equipe : [NOMBRE] - Outils principaux : [LISTE] - Premier projet : [DESCRIPTION] Pour chaque phase : - Objectifs (2-3) - Actions concretes (4-5) - Personnes a rencontrer - Jalons de validation - Resources / formations

Prompt 18 — Creer un reporting recrutement

A partir de ces donnees, genere un reporting recrutement pour le COMEX : Donnees du mois : - Postes ouverts : [NOMBRE] - Postes closes : [NOMBRE] - Time-to-fill moyen : [JOURS] - Candidatures recues : [NOMBRE] - Entretiens realises : [NOMBRE] - Offres emises : [NOMBRE] - Offres acceptees : [NOMBRE] - Pipeline actuel : [DETAILS] Genere : 1. Dashboard synthetique (texte) 2. 5 KPIs commentes (tendance vs mois precedent) 3. Alertes (postes a risque) 4. 3 recommandations actionnables

Prompt 19 — Construire une scorecard de poste

Cree une scorecard d'evaluation pour le poste de [POSTE]. Brief : [COLLER LE BRIEF] La scorecard doit contenir : - 4-5 criteres techniques (ponderes) - 3-4 soft skills - 2-3 criteres culture fit - Echelle de notation 1-5 avec descripteurs par niveau - Score minimum pour passer a l'etape suivante Format : tableau pret a copier-coller dans un Google Sheet.

Prompt 20 — Analyser les raisons de refus d'offre

Analyse ces retours de candidats ayant refuse nos offres sur les 6 derniers mois : [COLLER LES RETOURS / FEEDBACKS] Fournis : 1. Les 5 raisons de refus les plus frequentes (classees) 2. Tendances par rapport au marche 3. Actions correctives prioritaires (quick wins) 4. Recommandations a moyen terme 5. Benchmark : comment se situent nos offres vs le marche

Bonnes pratiques IA pour le recrutement

  • Toujours relire et adapter : L'IA genere un brouillon, pas un produit fini
  • Fournir du contexte : Plus le prompt est precis, meilleur est l'output
  • Ne pas copier de donnees sensibles : Anonymisez les CV avant de les coller
  • Tester plusieurs formulations : Iterer sur le prompt pour affiner
  • Documenter l'usage : Tracez quels outils IA vous utilisez (RGPD / AI Act)
  • Former l'equipe : Partagez les prompts qui fonctionnent au sein de l'equipe
